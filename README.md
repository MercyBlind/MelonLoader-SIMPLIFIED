# Setting MelonInstaller Up
1: Open the winrar folder **MelonLoader.Installer_2.rar**

2: Run **MelonLoader.Installer_2.exe**

3: Select your game by pressing **Select**

4: Go to your steam -> steamapps -> common -> Phasmophobia and select **Phamophobia**

5: Press **Ok** And click install.

6: Go back to your phasmophobia folder and you should see there are 3 folders that were created; MelonLoader, Mods, Logs

# Setting Mods up
1: Open **Bypass_Files_V7_Python.rar.rar**, Open Python fixed this for Gubna V7

2: Install NET Frame work and VC_redistx64.exe if you have not installed either.

3: Open "Modding Guides and Mods"

4: Open "MelonLoader" folder and select "1"

5: Drag Phasbypass.dll into your phasmophobia directory.

6: Open "2" and drag PhasBypassInjector.dll and drag it intoyour **Mods** Folder in your phasmophobia directory

7: Open "3" and go to your phasmophobia directory, open -> **MelonLoader -> Dependencies -> AssemblyGenerator** and drag and drop MelonLoader.AssemblyGenerator in it from the folder "3" in winrar, drag it into the folder and replace it.
